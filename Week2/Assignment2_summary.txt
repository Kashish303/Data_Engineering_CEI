🛒 ShopEase E-Commerce Database Analysis using SQL
📌 Project Overview

This project focuses on analyzing an E-Commerce database using SQL while applying real-world database concepts such as filtering, constraints, indexing, joins, aggregations, conditional logic, and transaction management.

The objective was not only to write SQL queries but also to understand how relational databases support business operations such as customer management, order processing, inventory tracking, sales reporting, and data integrity.

The project simulates a real-world e-commerce environment where customers place orders, products are managed through inventory, and transactions must remain consistent, accurate, and reliable.

🎯 Project Objectives

The primary goals of this project were:

Understand relational database design and relationships.
Retrieve and filter data using SQL queries.
Enforce data integrity using constraints.
Improve query performance using indexes.
Combine data from multiple tables using joins.
Generate business insights through aggregations and conditional logic.
Understand transaction management and ACID properties.
Translate business requirements into SQL-based solutions.
